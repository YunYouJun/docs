# @yunyoujun/docs

搭建文档（VitePress）常用的配置、组件、样式等。

## 使用

```bash [pnpm]
pnpm add -D @yunyoujun/docs
```

```bash [pnpm]
# 按需添加
pnpm add -D unocss unplugin-vue-components
# vitepress
pnpm add -D vitepress-plugin-group-icons @shikijs/vitepress-twoslash
```
